The Need
The need was for an updated website for this year’s lottery that kept the look and feel of print designs but didn’t move away from the original layout. There was also a need for a content management system to help organize and optimize the development process.
The Solution
The solution was to keep the basic structure of the website while ensuring the look and feel of the site was modern and fresh. Using white and subtle background graphics allowed for more vibrant colours to be used for CTA’s and web graphics. It was also important to keep the same functionality across all devices instead of cutting out features while still keeping the site’s components mobile friendly. The use of Wordpress made the transition from design to development smooth and simple.
