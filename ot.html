The Need
The need was for a webpage for Ottawa Tourism to act as a sales agent for conventions in Ottawa. Things like hotels, and convention meeting places were to be included in the web app. There was also a need for the sales agent to keep in the look and feel of Ottawa Tourism’s new website that is in development.
The Solution
The solution was to keep all the convention options in one place where the user can click to find out more information about the option they chose. The sales agent uses a pop-up dialog box for the information for each tile. The design of the sales agent is consistent with the design of Ottawa Tourism’s newest site which is currently still in development.
