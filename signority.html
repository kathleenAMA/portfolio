The Need
The need was for a new interface for an online e-signing web application. An interface that allows for a quick signing process, but enough options for the signer to customize the signing process to their document and needs.
The Solution
The solution was to keep the interface uncluttered and bright so that the options would not overwhelm the user. Giving an account summary and simple visual icons for signing choices on the dashboard page allows user to see all their options in front of them rather than navigating through menus to see their signing choices.
